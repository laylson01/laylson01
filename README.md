### 🌐 Network Automation Toolkit (Python)
Automação de tarefas de rede (ex.: parsing de configs, inventário, validações e relatórios).
- Stack: Python, Linux, (opcional: Netmiko/NAPALM), Pandas  
- Repo: https://github.com/SEUUSUARIO/network-automation-toolkit

### 📊 Network Logs Analyzer
Pipeline simples para analisar logs (erros recorrentes, horários de pico, alarmes) e exportar relatórios.
- Stack: Python, Pandas, SQL  
- Repo: https://github.com/SEUUSUARIO/network-logs-analyzer

### 🔗 API de Inventário / Documentação de Rede
API para cadastrar sites, links, equipamentos, VLANs e exportar documentação.
- Stack: Django, REST API, SQL  
- Repo: https://github.com/SEUUSUARIO/network-inventory-api
